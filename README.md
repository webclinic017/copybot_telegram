# Simple Telegram Trading Bot

Telegram signals will be traded automatically in real-time on the broker.

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

Create a `.env` file in this directory with these variables:

```bash
TELEGRAM_API_ID = ""
TELEGRAM_API_HASH = ""
TELEGRAM_CHAT = ""

APCA_API_KEY_ID = ""
APCA_API_SECRET_KEY = ""
APCA_API_BASE_URL = ""

METAAPI_TOKEN = ""
METAAPI_ACCOUNT_ID = ""
```
